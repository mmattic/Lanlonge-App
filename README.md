# Lanlonge App (蓝隆科技)

This is a directory and navigation web application for European cross-border sellers.

## Deployment

This project is configured to automatically deploy to **GitHub Pages** using GitHub Actions.

1. Push the code to the `main` or `master` branch.
2. Go to **Settings > Pages** in your GitHub repository.
3. Ensure **Source** is set to **GitHub Actions**.
4. The site will be live at `https://[your-username].github.io/Lanlonge-App/`.

## Local Development

1. `npm install`
2. `npm run dev`
